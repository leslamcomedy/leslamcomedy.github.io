# 如何贡献

这个知识库欢迎俱乐部所有演员一起补充！一个人的眼界有限，大家一起收集才能让资料越来越全。

## 怎么投稿？发微信就行（最简单）

发现好的视频、播客、书、文章、段子，**直接把链接发到俱乐部微信群，或者私发给主理人**，最好加一句放到哪个分类。不需要注册任何账号，不需要懂任何技术，剩下的交给主理人，一般一两天内就会出现在网站上。

## 收集什么内容？

- 脱口秀技巧类的视频、文章
- 优秀的专场和表演（带文稿的更好）
- 相关播客和书籍
- 演出信息、开放麦资讯
- 你自己写的好段子和创作心得

## 在 GitHub 上提议修改（会创建 Pull Request）

每个页面底部有「**提议修改此页（自动创建 Pull Request）**」链接。流程如下：

1. 点击链接，登录 GitHub
2. 在网页上直接改内容
3. 点提交时，GitHub 会**自动创建一个新分支**，并**打开一个 Pull Request（PR）**
4. 你**不需要**自己 push，也**不会**直接改到网站上
5. 主理人审核通过后合并 PR，网站才会自动更新（通常几分钟内）

::: tip 为什么用 Pull Request？
这样每次修改都有记录，主理人可以检查内容是否合适，避免误改或不当内容直接上线。
:::

::: warning 注意
「好的段子」栏目是从其他仓库自动同步的，**不能在这个网站仓库里直接改**。要改段子请在你的内容仓库里修改，或发微信给主理人。
:::
