# 如何贡献

这个知识库欢迎俱乐部所有演员一起补充！一个人的眼界有限，大家一起收集才能让资料越来越全。

## 没有 GitHub 账号？发微信就行

**这是给大多数成员准备的方式**，不需要注册任何账号，不需要懂编程。

发现好的视频、播客、书、文章、段子，按下面做：

1. 把**链接**发到俱乐部微信群，或者**私发给主理人**
2. 最好加一句：想放到哪个分类（比如「播客」「好的段子」「演出资讯」）
3. 如果是自己写的段子或心得，也可以直接把文字发过来

主理人收到后会帮你加到网站上，一般一两天内就能看到。你什么都不用管。

::: tip 举例
「主理人，这个播客不错，放播客栏目：https://...」

「这是我写的一段开场白，能放到好的段子吗？[粘贴文字]」
:::

## 收集什么内容？

- 脱口秀技巧类的视频、文章
- 优秀的专场和表演（带文稿的更好）
- 相关播客和书籍
- 演出信息、开放麦资讯
- 你自己写的好段子和创作心得

## 有 GitHub 账号？可以自己提修改（可选）

如果你会用 GitHub，每个页面底部有「**提议修改此页（自动创建 Pull Request）**」链接：

1. 点击链接，登录 GitHub
2. 在网页上直接改内容
3. 点提交时，GitHub 会**自动创建一个新分支**，并**打开一个 Pull Request（PR）**
4. 你**不需要**自己 push，也**不会**直接改到网站上
5. 主理人审核通过后合并 PR，网站才会自动更新（通常几分钟内）

::: tip 为什么用 Pull Request？
这样每次修改都有记录，主理人可以检查内容是否合适，避免误改或不当内容直接上线。
:::

::: warning 注意
「好的段子」栏目是从其他仓库自动同步的，**不能在这个网站仓库里直接改**。要改段子请在你的内容仓库里修改，或发微信给主理人。
:::
