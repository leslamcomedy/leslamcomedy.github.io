# 如何贡献

这个知识库欢迎俱乐部所有演员一起补充！一个人的眼界有限，大家一起收集才能让资料越来越全。

## 怎么投稿？发微信就行

发现好的视频、播客、书、文章、段子，**直接把链接发到俱乐部微信群，或者私发给主理人**，最好加一句放到哪个分类。不需要注册任何账号，不需要懂任何技术，剩下的交给主理人，一般一两天内就会出现在网站上。

## 收集什么内容？

- 脱口秀技巧类的视频、文章
- 优秀的专场和表演（带文稿的更好）
- 相关播客和书籍
- 演出信息、开放麦资讯
- 你自己写的好段子和创作心得

::: details 会用 GitHub 的朋友看这里（可选）
每个页面底部都有「在 GitHub 上编辑此页」链接，登录 GitHub 后可以直接修改页面内容并提交，由主理人审核后网站自动更新。不会用也完全没关系，发微信一样快。
:::
