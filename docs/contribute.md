# 如何贡献

这个知识库欢迎俱乐部所有演员一起补充！一个人的眼界有限，大家一起收集才能让资料越来越全。

## 怎么投稿？

发现好的视频、播客、书、文章、段子，**直接把链接或文字发到俱乐部微信群，或者私发给主理人**。

最好加一句想放到哪个分类，例如：

- 「这个播客不错，放播客栏目」
- 「这段开场白，能放到好的段子吗？」

不需要注册任何账号，不需要懂编程。主理人收到后会帮你加到网站上，一般一两天内就能看到。

## 收集什么内容？

- 脱口秀技巧类的视频、文章
- 优秀的专场和表演（带文稿的更好）
- 相关播客和书籍
- 演出信息、开放麦资讯
- 你自己写的好段子和创作心得

::: warning 注意
「好的段子」栏目是自动同步的，**不能在这个网站上直接改**。要改段子请在你的内容仓库里修改，或发微信给主理人。
:::
