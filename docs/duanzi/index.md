# 好的段子

俱乐部收集整理的优秀段子文稿，内容自动更新。

先读文稿分析结构和笑点设计，再配合[表演视频](/shows)看节奏处理，学习效果最好。

在左侧目录中选择要看的文稿。

## 排版说明

网站会**原样保留**你 md 文件里的换行格式：

- 一句一行（中间只有一个回车）→ 网页上也是一句一行
- 一段话一行（多个句子写在同一行）→ 网页上也是一整段
- 句与句之间有空行 → 网页上分成多个段落

同步过程**只复制文件，不改内容**。

## 如何添加新段子？

在你的内容仓库里，往「好的段子」文件夹新增或修改 `.md` 文件，然后：

1. **等自动同步**：每天北京时间中午 12:00 左右自动更新
2. **立刻同步**：打开 [Actions 页面](https://github.com/leslamcomedy/leslamcomedy.github.io/actions)，点「构建并部署到 GitHub Pages」→ Run workflow

你**不需要**动网站这个仓库，也不用 commit 段子文件到这里。
